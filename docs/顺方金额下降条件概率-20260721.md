# 顺方金额下降条件概率分析（2026-07-21）

## 方法

- 样本：全部桌型、庄龙+闲龙、episode 内 ≥2 局监测数据；连续两局对比，**上一局排除和局**
- 升降判定：严格比较（降=本局<上局，升=本局>上局，持平不参与判定）
- 结果分类：当前局开顺方=续、开反方=反、开和=和
- 概率区间：Wilson 95% CI

## 基准

全部符合条件的连续对 n=5527：**续 44.9% / 反 46.0% / 和 9.1%**
（与自然基准一致：P(B)≈50.7%、P(P)≈49.3% 混合后约 45/46，和率 ≈9.5%）

## 解读

1. **条件 1-3（n≥199）：与基准完全重合，无预测力。**
2. **条件 4、6 方向性偏高**（续 49.7%/50.8% vs 基准 44.9%，反 40.3%/38.1% vs 46.0%），
   但 95% CI 仍覆盖基准值，**统计上不显著**；样本量 159/118，需要约 3 倍样本才能判定。
   含义假设：双方金额都降（散户退潮）但顺方人数在升（新散户跟进顺方）→ 龙略倾向于续。
   这与"平台杀重钱方"假设方向相反，反而像自然从众行为的噪声。
3. 条件 5、7 n=35，区间过宽，无信息。
4. 综合既有证据链（断点无痕、最后一秒无痕、本条件概率），**池级资金/人数对结果的预测窗口依然为零**；
   条件 4/6 作为"待扩样弱线索"登记，不作为下注依据。

## 复现口径

- 顺方=episode.side（庄龙取 3001+3013，闲龙取 3002）；金额/人数取自结算时 bet_json 快照
- 排除：episode 内 <2 局、上一局为和局、对应池上局为 0 或含持平的对
