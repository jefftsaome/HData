# 顺方金额下降条件概率分析（2026-07-21）

## 方法

- 样本：全部桌型、庄龙+闲龙、episode 内 ≥2 局监测数据；连续两局对比，**上一局排除和局**
- 条件：顺方（长龙一方）池金额环比下降
- 结果分类：当前局开顺方=续、开反方=反、开和=和
- 概率区间：Wilson 95% CI

## 基准

全部符合条件的连续对 n=5563：续 45.2% / 反 45.7% / 和 9.1%
（与自然基准一致：P(B)≈50.7%、P(P)≈49.3% 混合后约 45/46，和率 ≈9.5%）

## 结论

**所有切片的置信区间都覆盖基准值——"顺方金额下降"及其与顺方人数/反方金额/反方人数的任意组合，
对下一局续/反/和均无预测力。** 此前用户关注的"人数未降但金额降"场景（n=647）同样落在基准上。

这与《极速闲龙断点分析-20260721》《最后一秒操控检验-20260721》结论一致：
池级资金/人数数据与牌局结果独立。至此"资金下降预兆断龙"这一假设可以**正式排除**。

## 复现口径

- 顺方=episode.side（庄龙取 3001+3013，闲龙取 3002）；金额/人数取自结算时 bet_json 快照
- "下降"=严格小于上局；"未降"=持平或上升
- 排除条件：episode 内 <2 局监测数据、上一局为和局、上局对应池为 0 的对
