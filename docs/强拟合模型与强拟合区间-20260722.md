# 强行拟合断龙模型 + 强拟合时间区间特征（2026-07-22）

> 数据：data/streak.db 全部长龙 episode 内相邻非和局对 10,625
> （基准反率 0.501）。采集进行中，为当时快照。
> 复现：`uv run python scripts/fit_break_model.py`
> 方法：束搜索枚举 ≤4 条件组合（93 个条件：4 项方向/金额人数倍率/龙长/
> 小时/龙向/桌型/在线变化/人均变化/顺方份额），最大化子集胜率。

## 结论（先看这里）

1. **强拟合天花板 = 噪声口袋。** 含小时条件、min_n=40 时最高胜率
   0.762（16时∧经典百家乐∧顺人均↓∧顺人↑，n=42）；Top15 规则 n 全部
   贴在 40-46 的 min_n 边界上——这是数据挖矿挖出噪声口袋的典型特征。
2. **约束稍紧，天花板立刻塌。** 禁用小时条件、min_n=100 后，最高胜率
   只有 0.630（主播百家乐∧顺额比>1.5∧反人↑，CI [0.532, 0.718]）。
3. **样本外验证全部打回原形**（07-20/21 拟合 → 07-22 验证）：
   - 全池：拟合 0.756 → 验证 0.571（n=7）；唯一"幸存"的一条
     0.732 → 0.679（n=28，CI [0.493, 0.821]）仍跨 0.5，
     且是 8 选 1 的最大值（选择偏差）；
   - 禁小时池：拟合 0.62-0.64 → 验证 0.41-0.54，全面崩塌至基准以下。
4. **强拟合时间区间不可采信。** 最强无时间规则逐小时拆胜率，
   每小时只剩 n=3-8；唯一"强区间"08h 胜率 0.875 但 n=8，
   其区间特征（在线 35 vs 95、总池 6 万 vs 10.7 万）随 n=8 一起失效。
5. **总结论**：在结算级公开数据（金额/人数/在线/龙长/桌型/小时）上，
   不存在能被强拟合出 70%+ 胜率的结构；任何样本内 70%+ 都是过拟合
   幻觉，样本外即消失。与此前全部检验（七条件、三局链式、路径解剖、
   扎堆检验）结论一致。残存活口仍是帧级「追龙鲸」信号
   （见《反方七条件概率+断龙路径解剖-20260722》）。
6. 本流程已固化为脚本，数据攒多后可直接重跑复核；若未来某规则在
   **样本外** CI 下沿站稳 0.70，才进入实弹讨论。
