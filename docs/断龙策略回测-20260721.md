# 断龙策略回测（2026-07-21）

## 策略规则（用户拟定）

龙局中，当前局满足以下全部条件则押反方（赌断龙）：
1. 顺方人数和金额**双双**突破上一局结算值，且首次双超发生在封盘前 5 秒内（任一指标 5 秒前已超则作废）；
2. 收盘时顺方人数/金额 ≤ 上局 3 倍；
3. 收盘时反方人数/金额 > 上局 且 ≤ 3 倍。

判定：开反=赢，开顺=输，开和=走水。

## 回测数据

全部龙 episode 相邻对（上局非和、上局四指标非零、有 ≥3 帧快照）：6,469 对。

判定分布：触发 1133（扣和 1025）｜5秒前已超 2020｜无双超 877｜反方未双超 606｜反方超3倍 744｜顺方超3倍 1084。

## 结果

| 组别 | n(扣和) | 断率 |
|---|---|---|
| **策略触发** | 1025 | **51.5% [48.5,54.6]** |
| 扣和基准（全体龙局对） | ~6400 | **50.6%** |
| 消融：早盘双超+终值满足（去掉5秒时机） | 209 | 53.1% [46.3,59.8] |
| 尾盘双超·终值不满足 | 2212 | 50.1% [48.1,52.2] |

表面 EV（定额 1 单位押反方，庄 0.95/闲 1.0 赔率）：+0.021/局。

## 结论：策略无优势

1. **比较基准修正**：策略和局走水，必须对比扣和断率 50.6%（而非含和的 45.8%）——
   修正后策略断率 CI 完全覆盖基准，**无超额**；
2. **消融实验**："5 秒内双超"时机条件零贡献（早盘双超组 53.1% ≈ 策略组），
   筛选力全部来自"双超+3倍内"这个静态条件，而它选出的就是断率自然基准的普通进行局；
3. **EV 为正的错觉**：来源于押反方时一半情况押闲（1:1 赔率）的赔率结构，
   闭眼每局押反方 EV 也 ≈0；策略未带来超额收益；
4. 与全项目证据链一致：池级资金/人数的任何组合条件都不携带结果信息。

## 方法论教训（重要）

- 回测策略必须**先定正确的对照基准**：含和/扣和口径差 5 个百分点，足以把"无效"误判为"有效"；
- 必须做**条件消融**：逐条去掉规则看贡献，避免把"筛选效应"误认为"信号"；
- EV 对比必须扣除赔率结构本身带来的基准 EV。
