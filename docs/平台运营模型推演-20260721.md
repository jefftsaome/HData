# 平台运营模型推演（2026-07-21）

> 目的：站在平台/赌徒/产品三个角度建立统一模型，指导后续信号发掘方向。
> 证据分级：【已证】有数据支撑；【弱线索】名义显著未钉死；【推演】逻辑推理无直接数据。

## 一、平台视角：它到底怎么挣钱

### 1. 收入结构（三层）

| 层 | 机制 | 占比估计 | 证据 |
|---|---|---|---|
| 规则抽水 | 庄免佣 5%、和局赔率差、B6 减半、侧注高水位 | 稳定基本盘 | 【已证】赔率结构 |
| 流水放大 | 数据展示制造"有据可依"感，提升下注频次/注额 | 放大器 | 【已证】方向性检验全阴性=维护可信度 |
| 选择性收割 | 点杀个体、蜜月期、分组推送、（可能的）变牌 | 增量利润 | 【弱线索】追龙鲸 27%、账号分组、大池少和 |

### 2. 核心经营逻辑：体面是最大的资产

- 平台运营十几二十年、全亚洲最大——**它的护城河是"看起来正规"**；
- 推论：任何会被统计抓到的作弊都不会做（我们 20+ 项聚合检验全阴性正是"严丝合缝"的直接证据）；
- 做手脚只能发生在三个隐蔽位：**个体级**（点杀，聚合数据隐形）、**账号级**（蜜月/分组，需玩家级数据）、**物理级**（变牌，协议数据不可见）；
- 这解释了用户观察到的"数据推送点数与实际开牌不一致"：如果存在，属于物理级操作，我们的数据维度天然无法证伪/证实。

### 3. 桌台组合管理（赌场楼层逻辑）

- 真人赌场早就按"楼层组合"管理：有的桌热、有的桌冷，整体收益率达标即可；
- 【推演】平台大概率同样按桌台/桌型做组合管理：部分桌台允许亏（引流桌、口碑桌），
  部分桌台承担利润指标（高流量极速桌）；
- 【推演】桌台利润目标若存在，其实现手段不是改单局结果（太显眼），
  而是**流量调度**：分组推送决定谁看到哪桌，把"肥羊"导向高收割桌；
- 可检验方向：对比两组账号可见桌台的后续走势质量（已在检验计划内）。

### 4. 真假数据混合

- 【已证】金额数据呈真实筹码面额结构（Benford 检验）——主流是真实下注；
- 【推演】注水更可能注在**人数字段**（成本低、难验证）：在线人数虚高制造繁荣感；
- 【已证】在线人数口径异常确实存在（在线<下注人数的倒挂、40 秒滞后）；
- 注意：如果掺假的是"人"而不是"钱"，那以人数为核心的指标（人均注额、人数占比）全部要带折扣用。

## 二、赌徒视角：心理收割地图

| 心理 | 触发场景 | 平台的利用 |
|---|---|---|
| 从众 | 人数/金额暴涨 | "大家都在买"=免费劝诱，数据展示即广告 |
| 跟大户 | 鲸入场播报 | 大户=方向图腾，追到就是被收割（追龙鲸 73% 断）【弱线索】 |
| 斩龙冲动 | 长龙 7+ | "该反了"的赌徒谬误，平台只需让龙自然生死即可赚两边的钱 |
| 差一点心理 | 断龙、和局 | "就差一点"促进立刻再押 |
| 损失追回 | 连输后 | 加注=流水放大，且进入被点杀候选池 |
| 控制幻觉 | 全部数据展示 | "我看得懂数据"=下注频次提升的根因 |

## 三、产品视角：漏斗与防御

```
拉新  → 蜜月期让新人先赢（建立"我能赢"信念）【推演，多人反馈】
激活  → 分组推送"好看的桌台"+好路标签，降低开局门槛【分组已证】
留存  → 极速节奏+倒计时压迫+差一点心理，提升局数/小时
变现  → 规则抽水为主，点杀/收割为辅【聚合无痕=收割极克制】
防御  → 5局踢出（反观察）、分组推桌（反比较）、数据只给池级（反透视）【已证】
```

## 四、对我们策略发掘的指导

1. **聚合公共数据已无矿**——继续挖是负期望值劳动；
2. 剩余三个未封死方向按优先级：
   a. 账号级信号（蜜月期/点杀）——需要真实下注的标注实验（朋友直觉实验）；
   b. 流量调度证据（两组账号桌台质量对比）——纯观察可做，性价比高；
   c. 人数字段注水指纹（在线/下注比异常模式）——纯观察可做；
3. 任何声称 70%+ 胜率的策略，若只基于池级公开数据，数学上已证伪，不必再测。
